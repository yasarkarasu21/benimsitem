# Kişisel Tanıtım Sitesi (Basit Şablon)

Bu proje, Bootstrap5, HTML5, CSS3 ve sade JavaScript kullanılarak oluşturulmuş bir kişisel tanıtım sitesi şablonudur.

Dosyalar
- `index.html` — Ana sayfa; bölümler: Hakkımda, Bilgilerim, Portföy, Eğitimlerim, Sertifikalarım, İletişim.
- `css/styles.css` — Özel stiller.
- `js/script.js` — Smooth scroll, form simülasyonu, modal doldurma.

Nasıl çalıştırılır
1. `index.html` dosyasını tarayıcıda açın (çift tıkla veya tarayıcıya sürükleyin).
2. Yerel düzenleme için metin editörü ile `index.html` içindeki yer tutucu metinleri (ad, iletişim bilgileri, portföy maddeleri) güncelleyin.

Notlar
- Bu bir statik şablondur; iletişim formu gerçek bir sunucuya göndermez — form gönderimi sadece simüle edilir.
- Bootstrap ve diğer kütüphaneler CDN üzerinden yüklenir; çevrimdışı kullanım için ilgili dosyaları indirip referansları güncelleyin.
